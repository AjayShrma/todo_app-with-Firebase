# todo_app-with-Firebase

# Todo app

Todo is a CRUD application. In which you can CRUD your list of tasks.  

# New Features!

  - Directly delete Todo by cross button
  - Open Todo by a simple tap 


You can also:
  - Easly edit.
  - Can acess data even after deleting app.
